## Feature Improvement Plan

### Next Sprint: Exercise + Break Integration

**Goal:** Fuse eye exercises with break reminders. Recommend 1-2 exercise sessions per day based on screen time.

#### Design Concept

```
Screen time < 2h  → Suggest 1 exercise session
Screen time 2-4h  → Suggest 2 exercise sessions  
Screen time > 4h  → Strongly recommend 2+ sessions

Trigger: During macro break (60min) or mandatory break (120min),
         offer "Do exercises" button alongside the countdown.
```

#### UX Flow (Proposed)

```
Break overlay appears (existing 20s/5min countdown)
    │
    ├── [Current] Auto-countdown + skip button
    │
    └── [NEW] If exercise recommended:
            Show "Start Eye Exercises" button below countdown
            Badge: "Today: 0/2 sessions completed"
            │
            └── Tap → Launch exercise session (existing UI)
                      with AUDIO GUIDANCE (new)
```

#### Audio Guidance for Exercises (Priority)

```
Exercise step transition:
  1. Gentle chime (system sound) at step change
  2. AVSpeechSynthesizer reads instruction in Chinese
     e.g. "向上看，保持三秒" (Look up, hold 3 seconds)
  3. Countdown beeps for last 3 seconds of each step

Exercise completion:
  - Celebration sound (ascending chime)
  - "做得好！眼保健操完成" (Well done! Exercise complete)
```

#### Implementation Steps

1. **SoundManager: Add TTS capability**
   - AVSpeechSynthesizer with zh-CN voice
   - Queue-based: speak instruction, then wait for exercise duration
   - Volume respects existing settings

2. **ExerciseSessionView: Wire audio to step transitions**
   - On exercise start: speak exercise name + first instruction
   - On step change: speak next instruction
   - On exercise complete: celebration sound
   - On session complete: summary speech

3. **BreakScheduler: Track exercise sessions**
   - `exerciseSessionsToday: Int`
   - `recommendedExerciseSessions: Int` (computed from screen time)
   - Persist in daily data

4. **FullScreenOverlayView: Add exercise button**
   - Show during macro/mandatory breaks
   - Display "Today: X/Y exercises" badge
   - Button launches exercise session

5. **ExerciseSessionView: UI polish**
   - Larger step indicator
   - Better animation transitions
   - Audio on/off toggle

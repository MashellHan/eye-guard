## 1. Menu Bar Icon & Popover

### 1.1 Menu Bar States

```
┌──────────────────────────────────────────────────────────┐
│  Menu Bar                                         👁 17:42 │
└──────────────────────────────────────────────────────────┘

States:
  👁 17:42   → Normal (green tint): 17m 42s until next break
  👁  2:15   → Warning (yellow tint): break approaching
  👁  0:00   → Alert (red tint): break overdue
  👁  IDLE   → Dimmed (gray): user is idle, timer paused
  👁  ⏸      → Paused: breaks disabled by user
```

## 6. Visual Design Notes

### 6.1 Color Palette

```
Primary:       #007AFF  (macOS system blue)
Success:       #34C759  (system green)
Warning:       #FF9500  (system orange)
Danger:        #FF3B30  (system red)
Background:    System background (adapts to dark/light mode)
Text:          System label color (adapts to dark/light mode)
Overlay bg:    #000000 @ 60% opacity (Tier 3)
Tier 2 bg:     NSVisualEffectView .hudWindow material
```

### 6.2 Typography

```
Menu bar timer:  SF Mono, 12pt, medium weight
Popover title:   SF Pro, 16pt, semibold
Popover body:    SF Pro, 13pt, regular
Countdown large: SF Pro Rounded, 48pt, bold (Tier 2)
Countdown huge:  SF Pro Rounded, 72pt, bold (Tier 3)
Health score:    SF Pro Rounded, 20pt, bold
```

### 6.3 Iconography

```
Menu bar icon:   SF Symbol "eye" (adapts to menu bar style)
Break states:    eye, eye.slash, eye.trianglebadge.exclamationmark
Notification:    Custom eye icon with subtle animation
App icon:        Stylized eye with shield motif (protection theme)
```

### 6.4 Animation Guidelines

```
Popover open:         0.2s ease-in-out (system default)
Tier 2 appearance:    0.3s fade-in + slide from right
Tier 2 dismissal:     0.2s fade-out + slide to right
Tier 3 appearance:    0.5s fade-in, blur ramps up over 0.3s
Tier 3 dismissal:     0.3s fade-out
Progress ring:        Continuous smooth animation (CADisplayLink)
Timer countdown:      1s tick with subtle scale pulse
Score change:         0.5s spring animation on number change
```

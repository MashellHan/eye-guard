### Tier 2 Behavior Notes

```
Position: Top-right, 20pt margin from screen edges
Animation: Slides in from right edge (0.3s ease-in-out)
Backdrop: NSVisualEffectView with .hudWindow material
Persistence: Stays until dismissed or break detected
Draggable: Yes, user can reposition
Close: Click "I'll rest now" or idle 20+ seconds
```

### Menu Bar Icon States

```
State           Icon      Color     Animation
─────────────────────────────────────────────
Active          👁        Green     Static
Break Due       👁        Yellow    Slow pulse (2s)
Overdue         👁        Red       Fast pulse (0.5s)
On Break        👁        Blue      Breathing (3s)
Paused          👁        Gray      Static
```

## Accessibility

- All interactive elements have accessibility labels
- VoiceOver support for all notifications and overlays
- Keyboard navigation in preferences and popover
- High contrast mode support
- Reduced motion respects `NSWorkspace.shared.accessibilityDisplayShouldReduceMotion`
- Tier 3 overlay announces via VoiceOver before appearing

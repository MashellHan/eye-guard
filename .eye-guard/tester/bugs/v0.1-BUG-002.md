# BUG-002: `recalculateHealthScore` uses only current session, not daily totals

**Bug ID**: v0.1-BUG-002
**Title**: Health score calculation uses current session duration instead of accumulated daily totals
**Severity**: HIGH
**Component**: BreakScheduler
**File**: `EyeGuard/Sources/Scheduling/BreakScheduler.swift`, lines 229-243

## Steps to Reproduce

1. Start the app and use the computer for 2 hours (total screen time: 2h)
2. Go idle for 5 minutes (triggers `handleActivityResumed`, resets `currentSessionDuration` to 0)
3. Resume working for 30 minutes
4. Take a break
5. Observe that `recalculateHealthScore()` passes `currentSessionDuration` (30 min) as both `totalScreenTime` and `longestContinuousSession`

## Expected Behavior

Health score should reflect the full day's usage:
- `totalScreenTime` should be the sum of all sessions (2h + 30m = 2.5h)
- `longestContinuousSession` should be the longest single session (2h)

## Actual Behavior

```swift
private func recalculateHealthScore() {
    // ...
    let score = calculator.calculate(
        breakEvents: todayBreakEvents,
        totalScreenTime: currentSessionDuration,         // BUG: only current session
        longestContinuousSession: currentSessionDuration  // BUG: only current session
    )
}
```

After an idle period resets the session, the health score calculation loses all prior session data. The score will be artificially high because it only sees the short current session, not the full day's accumulated usage.

Additionally, `longestContinuousSession` is always set to `currentSessionDuration`, so it can never reflect a previous session that was longer.

## Impact

- Health scores are inaccurately inflated after idle/resume cycles
- The "Total Screen Time" component of the score (20 pts) never penalizes users with many short sessions adding up to excessive total time
- The "Continuous Use Discipline" component (30 pts) forgets long sessions from earlier in the day

## Suggested Fix

Track accumulated daily metrics:

```swift
private var totalDailyScreenTime: TimeInterval = 0
private var longestDailySession: TimeInterval = 0

func handleActivityResumed() {
    guard !isPaused else { return }
    totalDailyScreenTime += currentSessionDuration
    longestDailySession = max(longestDailySession, currentSessionDuration)
    sessionStartTime = .now
    currentSessionDuration = 0
}

private func recalculateHealthScore() {
    let accumulatedScreenTime = totalDailyScreenTime + currentSessionDuration
    let longestSession = max(longestDailySession, currentSessionDuration)
    let score = calculator.calculate(
        breakEvents: todayBreakEvents,
        totalScreenTime: accumulatedScreenTime,
        longestContinuousSession: longestSession
    )
    currentHealthScore = score.totalScore
}
```

# v1.1 Implementation Notes — Mascot Expressions + Click Interactions

## Summary

Enhanced mascot with richer facial expressions (eyebrows + mouth for every state), mouse-following pupil tracking, double-click toggle, and an expanded right-click context menu.

## Changes

### MascotView.swift (rewritten)
- **Eyebrows** drawn with custom `Shape` paths for each state:
  - `HappyBrow` — gentle upward curve (happy, celebrating)
  - `ConcernedBrow` — angled down, worried (concerned), paired left/right
  - `RaisedBrow` — high arch, surprised (alerting), paired left/right
  - `FocusedBrow` — straight angled line (exercising), paired left/right
  - Sleeping: relaxed `HappyBrow` with low opacity
- **Mouth** expressions for each state:
  - Happy/Celebrating: `HappyMouth` smile arc (existing, kept)
  - Concerned: small `Circle` stroke (worried "O" shape)
  - Alerting: larger `Ellipse` stroke (open surprise mouth)
  - Sleeping: `SleepMouth` gentle arc with low opacity
  - Exercising: determined flat `RoundedRectangle` line
- **Expression extras**: alerting now shows dual exclamation marks on both sides
- **New Shape types**: `HappyBrow`, `ConcernedBrow`, `RaisedBrow`, `FocusedBrow`, `SleepMouth`

### MascotContainerView.swift
- **Hover tracking state**: Added `isHoverTracking`, `hoverPupilOffset` to `MascotViewModel`
- **`updateHoverPupil(mousePosition:mascotCenter:)`**: Calculates pupil offset from mouse position relative to mascot center, max 4pt displacement, 200pt tracking range
- **`stopHoverTracking()`**: Resets hover state
- **Idle look-around**: Now skips random movement when hover tracking is active
- **Double-click**: Added `.onTapGesture(count: 2)` to toggle speech bubble
- **Single click**: Kept, still toggles bubble
- **Context menu expanded**: Added "Generate Report" and "Quit EyeGuard" items
- **`onGenerateReport` callback**: New optional callback for report generation

### MascotWindowController.swift
- **Mouse tracking (v1.1)**: Added `startMouseTracking()` with 10fps polling task
  - Reads `NSEvent.mouseLocation` (global mouse position)
  - Calculates mascot center from `window.frame.midX/midY`
  - Calls `viewModel.updateHoverPupil()` to move pupil toward cursor
- **`stopMouseTracking()`**: Cancels polling task on hide
- **`handleGenerateReport()`**: Generates daily report and opens folder in Finder
- **`viewModel` stored**: Now keeps a reference to the view model for mouse tracking access

### build-app.sh
- Updated version to `1.1.0`, build `11`

## Architecture — Mouse Tracking Flow

```
NSEvent.mouseLocation (10fps poll)
       ↓
MascotWindowController.startMouseTracking()
       ↓
MascotViewModel.updateHoverPupil(mousePosition, mascotCenter)
       ↓
  Calculate angle + distance → clamp to 4pt max
       ↓
  pupilOffset animated → MascotView pupil moves
```

## Interaction Summary

| Gesture | Action |
|---------|--------|
| Single click | Toggle speech bubble |
| Double-click | Toggle speech bubble |
| Long press | Toggle speech bubble |
| Right-click | Context menu (Take Break, Snooze, Report, Settings, Quit) |
| Hover near | Pupil follows cursor |
| Drag | Move mascot window |

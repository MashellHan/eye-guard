# v0.9 — Eye Guard Mascot (护眼精灵)

## Summary

Implemented a cute, round eyeball mascot character that floats on screen and reacts to the user's eye health behavior. The mascot is drawn entirely with SwiftUI shapes — no image assets required.

## Architecture

```
EyeGuard/Sources/Mascot/
├── MascotState.swift           — 7-state enum (idle, happy, concerned, alerting, sleeping, exercising, celebrating)
├── MascotView.swift            — SwiftUI shape-based character (body, iris, pupil, arms, legs, expressions)
├── MascotAnimations.swift      — Centralized animation parameters (timing, amplitude, patterns)
├── MascotContainerView.swift   — Container + MascotViewModel (animation orchestration + scheduler sync)
├── MascotWindowController.swift — NSWindow setup (borderless, floating, transparent, draggable)
└── SpeechBubbleView.swift      — Rounded bubble with triangle tail for messages
```

## Character Design

- **Body**: 64×64 pt white circle with subtle shadow (the eyeball)
- **Iris**: 34 pt gradient circle, color changes by state (blue/green/orange/red/indigo/teal/purple)
- **Pupil**: 16 pt black circle with white shine highlight
- **Arms**: Small gray rounded rectangles with circle hands, wave during alerting
- **Legs**: Two thin gray lines below body
- **Expressions**: Happy mouth (arc), concerned brow (squiggle), exclamation mark (alerting), 💤 (sleeping), ✨⭐ (celebrating)

## State Machine

| State | Trigger | Animations | Iris Color |
|-------|---------|-----------|------------|
| idle | Default | Breathing, blinking, look-around | Blue |
| happy | Health score ≥80 | Smile expression | Green |
| concerned | Break approaching (<2 min) or score <50 | Worried brow | Orange |
| alerting | Break due | Bouncing, arm waving, speech bubble | Red |
| sleeping | 10 PM – 6 AM | Closed eyes, 💤 emoji | Indigo |
| exercising | During break | Pupil follows exercise pattern | Teal |
| celebrating | After break taken | Bouncing, sparkle effects | Purple |

## Floating Window

- `NSWindow` with `.borderless` style mask
- `.floating` level — stays above other windows
- `.clear` background, no shadow — only the mascot is visible
- `.isMovableByWindowBackground = true` — user can drag it anywhere
- `.canJoinAllSpaces, .stationary` — visible on all desktops
- Default position: bottom-right corner, 20 pt margin

## Speech Bubble Messages

- Launch: "Hi! 我是护眼精灵 👋"
- Break approaching: "快要到[Break Name]了…"
- Low health score: "记得休息哦 👀"
- Good compliance: "做得好！继续保持 💪"

## Integration

- `MascotWindowController` stored on `AppDelegate` as static property
- Created in `EyeGuardApp.body.onAppear` with scheduler reference
- `MascotContainerView` polls `BreakScheduler` every 2 seconds for state updates
- Cleaned up in `applicationWillTerminate`

## Animation System

- **Breathing**: Scale 0.96–1.0, 2.5s cycle (always active)
- **Blinking**: Random 2–5s interval, 0.15s duration
- **Bouncing**: ±6pt vertical, 0.4s cycle (alerting/celebrating)
- **Arm waving**: ±30° rotation, 0.5s cycle (alerting)
- **Eye exercise**: 9-point pattern (up/right/down/left/diagonals/center), 0.8s per step
- **Idle look-around**: Random ±4pt pupil drift every 4s

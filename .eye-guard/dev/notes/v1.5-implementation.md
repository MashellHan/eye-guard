# v1.5 Implementation Notes — Color Balance Suggestions (颜色平衡建议)

## Date: 2026-04-15

## Summary
Added periodic screen color analysis that captures a small screenshot every 5 minutes, analyzes the dominant color family using pixel sampling, and suggests complementary colors to reduce eye strain. Color suggestions appear in the mascot's speech bubble during breaks with bilingual messages.

## Files Created
- `EyeGuard/Sources/Analysis/ColorAnalyzer.swift` — Observable singleton for periodic screen color analysis using CGWindowListCreateImage, grid-based pixel sampling, HSV-based color family classification
- `EyeGuard/Sources/Analysis/ColorSuggestionView.swift` — SwiftUI views for displaying color suggestions with colored circles and suggestion text
- `EyeGuard/Tests/ColorAnalyzerTests.swift` — 15 tests covering color families, complementary colors, suggestions, and state management

## Files Modified
- `EyeGuard/Sources/Mascot/MascotContainerView.swift` — Shows color suggestion when break ends (celebrating state)
- `EyeGuard/Sources/App/EyeGuardApp.swift` — Starts ColorAnalyzer on app launch
- `EyeGuard/Sources/Utils/Logging.swift` — Added colorAnalysis logger

## Color Analysis Pipeline
| Step | Description |
|------|-------------|
| 1. Capture | CGWindowListCreateImage captures main display |
| 2. Sample | Grid sampling every 50th pixel for efficiency |
| 3. Classify | RGB → HSV conversion, hue-based color family classification |
| 4. Detect | Most frequent non-neutral color family = dominant |
| 5. Suggest | Complementary color family with bilingual message |

## Architecture
- `ColorAnalyzer` is a `@MainActor @Observable` singleton
- Screen capture via `CGWindowListCreateImage` (deprecated warning accepted per task spec)
- Pixel analysis uses direct `CGContext` bitmap rendering — no Metal, no ML
- Color classification: RGB → HSV, hue-range-based family detection
- Saturation threshold (>0.15) filters out neutral/grayscale pixels
- Analysis runs every 300 seconds (5 minutes) on a background task
- Color history tracks last 12 entries (1 hour rolling window)
- Suggestions dictionary has 3+ messages per color family
- `ColorSuggestionView` and `InlineColorSuggestion` are reusable SwiftUI components
- Integration: color suggestion shown after break completion in MascotContainerView

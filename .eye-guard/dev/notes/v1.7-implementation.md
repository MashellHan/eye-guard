# v1.7 Implementation: Dashboard View + Historical Charts

## Date: 2026-04-15

## Summary

Added a full Dashboard window with three tabs (Today, History, Breakdown) using
SwiftUI Charts for data visualization. Historical data loaded from persisted JSON
files with aggregation for the last 7/30 days.

## Files Created

### `EyeGuard/Sources/Dashboard/HistoryManager.swift`
- `DailySummary` struct with computed properties (screenTimeHours, breakCompliancePercent)
- `loadHistory(days:)` — async loader reading YYYY-MM-DD.json files via DataPersistenceManager
- `loadToday()` — convenience for current day data
- `averageHealthScore()`, `totalScreenTime()`, `totalBreaksTaken()` aggregation methods
- Health score computed from scoreHistory last value or HealthScoreCalculator fallback

### `EyeGuard/Sources/Dashboard/DashboardView.swift`
- **Today tab**: Large circular gauge (DashboardGauge), 6 stat cards in grid layout,
  active session info with next break countdown
- **History tab**: Segmented picker for 7/30 days range:
  - `BarMark` chart for daily screen time (color coded: blue < 6h, orange < 8h, red > 8h)
  - `LineMark` + `PointMark` chart for health score trend (0-100 scale)
  - Summary stats row (avg score, total breaks, total screen time)
- **Breakdown tab**: Score component bars with explanations, `SectorMark` pie chart
  with inner radius donut style, total score display
- Supporting views: `DashboardGauge`, `DashboardStatCard`, `SummaryStatView`,
  `BreakdownComponentRow`
- Async data loading with ProgressView and ContentUnavailableView fallbacks

### `EyeGuard/Sources/Dashboard/DashboardWindowController.swift`
- `@MainActor` singleton window controller
- Standard NSWindow with title bar (titled, closable, resizable, miniaturizable)
- Reuses existing window if already visible (brings to front)
- Hosts DashboardView via NSHostingView

### `EyeGuard/Tests/DashboardTests.swift`
- 7 tests for HistoryManager aggregation methods
- Tests for DailySummary computed properties (screenTimeHours, breakCompliancePercent)

## Architecture

```
MenuBarView "Dashboard..." button
    └── DashboardWindowController.shared.showDashboard(scheduler:)
            └── NSWindow with NSHostingView(DashboardView)
                    ├── Today tab: Live data from BreakScheduler
                    ├── History tab: HistoryManager.loadHistory()
                    │       └── DataPersistenceManager.load(for: date)
                    │               └── ~/EyeGuard/data/YYYY-MM-DD.json
                    └── Breakdown tab: BreakScheduler.currentBreakdown
```

## Charts Used

| Chart Type | Framework Component | Purpose |
|---|---|---|
| Bar chart | `BarMark` | Daily screen time (hours) |
| Line chart | `LineMark` | Health score trend over time |
| Point chart | `PointMark` | Score data points with color coding |
| Donut chart | `SectorMark` | Score component breakdown |
| Gauge | Custom `DashboardGauge` | Current health score (0-100) |

## Data Flow

1. `DataPersistenceManager` saves `DailyData` JSON every 5 minutes
2. `HistoryManager.loadHistory(days:)` reads JSON files for past N days
3. Each file is aggregated into a `DailySummary` for charting
4. SwiftUI Charts renders `BarMark`, `LineMark`, `SectorMark` from summaries
5. Today tab reads live from `BreakScheduler` observable properties

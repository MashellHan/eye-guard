# v1.2 Implementation Notes — Eye Exercise Animations (眼保健操)

## Date: 2026-04-14

## Summary
Added guided eye exercise routines with animated instructions and mascot integration. Users can start exercises during break time or from the mascot's context menu.

## Files Created
- `EyeGuard/Sources/Exercises/EyeExercise.swift` — Exercise enum with 5 exercises, bilingual names/instructions, pupil patterns
- `EyeGuard/Sources/Exercises/ExerciseView.swift` — Individual exercise view with type-specific animations
- `EyeGuard/Sources/Exercises/ExerciseSessionView.swift` — Full session orchestrator (intro → exercises → completion)

## Files Modified
- `EyeGuard/Sources/Notifications/BreakOverlayView.swift` — Added "Start Eye Exercises" button to break overlay
- `EyeGuard/Sources/Mascot/MascotContainerView.swift` — Added "Eye Exercises" context menu item + callback
- `EyeGuard/Sources/Mascot/MascotWindowController.swift` — Exercise window management + mascot state transitions
- `EyeGuard/Sources/Utils/Logging.swift` — Added exercise logger

## Exercises Implemented
| Exercise | Duration | Animation Type |
|----------|----------|---------------|
| Look Around (上下左右看) | 40s | Directional arrows |
| Near-Far Focus (远近调焦) | 30s | Growing/shrinking circle |
| Circular Motion (转圈) | 30s | Orbiting dot with trail |
| Palm Warming (掌心热敷) | 30s | Glowing warmth effect |
| Rapid Blink (快速眨眼) | 20s | Blinking eye + counter |

## Integration Points
- Break overlay offers "Start Eye Exercises" as alternative to simple countdown
- Mascot context menu: "Eye Exercises 眼保健操"
- Mascot enters `.exercising` state during exercises, pupil follows pattern
- Mascot celebrates on completion
- Exercise session opens in a floating window (centered, borderless, draggable)

## Architecture
- `EyeExercise` enum holds all exercise data (duration, instructions, pupil patterns)
- `ExerciseView` handles per-exercise animation with `Timer`-driven state updates
- `ExerciseSessionView` manages session lifecycle: intro → exercise sequence → completion
- `MascotWindowController` manages the exercise window lifecycle

# v1.3 Implementation Notes — Medical Tips Carousel (护眼小贴士)

## Date: 2026-04-14

## Summary
Added a rotating medical tips system with 25 evidence-based eye health tips from AAO, WHO, and OSHA guidelines. Tips are bilingual (English/Chinese), appear in the mascot's speech bubble every 30 minutes, and are included in daily reports as "Tip of the Day."

## Files Created
- `EyeGuard/Sources/Tips/EyeHealthTip.swift` — Tip model struct with bilingual content and source citation
- `EyeGuard/Sources/Tips/TipDatabase.swift` — Static database of 25 evidence-based tips
- `EyeGuard/Sources/Tips/TipBubbleView.swift` — Dedicated tip bubble view with source citation and navigation

## Files Modified
- `EyeGuard/Sources/Mascot/MascotContainerView.swift` — Added 30-min tip rotation timer + "Show Eye Tip" context menu callback
- `EyeGuard/Sources/Mascot/MascotWindowController.swift` — Added `handleShowTip()` + wired `onShowTip` callback
- `EyeGuard/Sources/Reporting/DailyReportGenerator.swift` — Added "Tip of the Day" section using deterministic date-based tip selection
- `EyeGuard/Sources/Utils/Logging.swift` — Added tips logger

## Integration Points
- **Mascot speech bubble**: Random tip every 30 minutes (only in idle/happy states)
- **Context menu**: "Show Eye Tip 护眼贴士" for on-demand tip display
- **Daily report**: "Tip of the Day" section using deterministic date-based selection
- **TipDatabase.tipOfTheDay(for:)**: Day-of-year modulo for consistent daily tip

## Architecture
- `EyeHealthTip` is a simple `Identifiable, Sendable` struct
- `TipDatabase` is a static enum with `tips` array, `randomTip()`, `tipOfTheDay(for:)`, `nextTip(after:)`
- `TipBubbleView` is a reusable SwiftUI component (prepared for future standalone usage)
- Tip rotation integrated into existing `startStateSync()` tick loop to avoid extra timers

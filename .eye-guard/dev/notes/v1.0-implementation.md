# v1.0 Implementation Notes — Mascot Break Reminders

## Summary

Made the mascot the PRIMARY break reminder mechanism. When it's time for a break, the mascot animates excitedly and shows a speech bubble with Chinese-language messages.

## Changes

### MascotContainerView.swift (rewritten)
- **Break event wiring**: Monitors `scheduler.breaksTakenToday` and `scheduler.breaksSkippedToday` for changes to trigger celebrations and concern states
- **Break-in-progress detection**: Watches `scheduler.isBreakInProgress` to transition to `.exercising` state with pupil movement animation
- **Alerting on break due**: When `scheduler.timeUntilNextBreak <= 0`, mascot transitions to `.alerting` with type-specific Chinese messages:
  - Micro: "👁️ 该休息眼睛了！看看远处20秒"
  - Macro: "☕ 休息一下吧！站起来活动5分钟"
  - Mandatory: "🚶 该站起来活动了！休息15分钟"
- **Celebration after break**: `.celebrating` state for 5 seconds with "👏 做得好！眼睛感觉好多了吧", then transitions to `.happy`
- **Concern on skip**: `.concerned` state with "😢 跳过休息了...眼睛会累的"
- **Late night mode**: After 10 PM, `.sleeping` state with "🌙 太晚了，早点休息吧"
- **Click interactions**: Single click toggles speech bubble, long-press toggles bubble, context menu for Take Break / Snooze / Settings

### MascotAnimations.swift
- Added sleeping sway parameters (`sleepSwayAmplitude: 4pt`, `sleepSwayDuration: 3s`)
- Added celebrating pulse parameters (`celebrateScaleMin: 0.95`, `celebrateScaleMax: 1.08`)
- Added celebrating rotation (`celebrateRotation: 8°`)
- Added alert bounce parameters (`alertBounceAmplitude: 10pt`, `alertBounceDuration: 0.35s`)
- Added `celebrationDisplayDuration: 5s`

### MascotViewModel (in MascotContainerView.swift)
- Added `swayOffset`, `celebrateScale`, `celebrateRotation` animation properties
- Added `startSleepingSway()` — slow horizontal oscillation
- Added `startCelebrating()` — scale pulse + rotation wiggle + bounce
- Added `startAlertBouncing()` — larger amplitude bounce for alerting
- Added `toggleBubble()` for click interaction

### MascotWindowController.swift
- Added `onTakeBreak` and `onSnooze` callback wiring to MascotContainerView
- Added `handleTakeBreak()` — triggers immediate break via scheduler
- Added `handleSnooze()` — snoozes via NotificationManager

### MascotView.swift
- Replaced single 💤 emoji with floating "z Z" text for sleeping state

### BreakScheduler.swift
- Added `isBreakInProgress: Bool` and `activeBreakType: BreakType?` state
- `takeBreakNow()` now sets break-in-progress and auto-ends after break duration
- Added `endBreak()` method
- `resetDaily()` clears break-in-progress state

### build-app.sh
- Updated version to `1.0.0`, build `10`

## Architecture

```
BreakScheduler → (state changes) → MascotContainerView.startStateSync()
                                         ↓
                                    MascotViewModel.transition(to:)
                                         ↓
                                    MascotView (visual updates)
                                    SpeechBubbleView (messages)
```

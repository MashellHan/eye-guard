# EyeGuard

Medical-grade eye health guardian for macOS.

Smart screen time monitoring with evidence-based break reminders following AAO 20-20-20 rule.

## Features (v0.1)
- Menu bar app with session timer
- Activity monitoring (keyboard/mouse)
- 20-20-20 micro-break reminders
- Hourly macro-break reminders
- Idle detection
- Medical evidence-based timing

## Build
```bash
swift build
swift run EyeGuard
```

## Medical Basis
- AAO 20-20-20 Rule: Every 20 min, look 20 feet away for 20 seconds
- WHO: Regular breaks for computer workers
- Break intervals based on peer-reviewed CVS research
